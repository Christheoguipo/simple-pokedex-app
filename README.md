## Project description:
  - A simple Pokedex App.

## How to use:
  - Click on a Pokemon to show its details.
  - Filter out Pokemons using the Search Bar. 

## Project dependencies:
  - jQuery v3.3.1
  - Popper.js v1.14.7
  - Bootstrap v4.3.1

## Which API the project uses?
  - [Pokemon API](https://pokeapi.co/api/v2/pokemon/)
